Kendo UI Examples

A collection of JayData examples with Kendo UI.

Netflix

Netflix movies demo application using JayData and Sencha Touch 2 together. JayData is perfect for data while Sencha is great in the UI.

Flower Shop

A simple e-shop HTML5 app created with Jaydata, JayStorm, KnockoutJS and jQuery.
Check out the hands on lab video!

Knockout.js example

Demonstrating easy data binding using JayData and Knockout.js together. Properties of the requested entities will be observable through a single toArray call if ko.observableArray is passed as parameter.

Knockout Dynamic Queries

With the JayData module for Knockout.js you can achieve fully dynamic queries. Where you use an observable as a query parameter the query will be automatically executed when the observable property changes.

Knockout Dynamic Queries filter

A Northwind example demonstrating filter features using the dynmaic queries of the JayData Knockout module. The application still remains only a few lines of code.

Yahoo Query language

With the Yahoo Query Language (YQL) you can query the content of the world wide web. News and RSS feeds, pure HTML content and Geo data, almost everything is queryable through the YQL provider. Here you can find some samples to see for yourself!
Examples: Html search, GEO based autocomplete, RSS example, RSS advanced example, Feed

Newsreader

CRUD (Create-Read-Update-Delete) example using OData provider. Managing categories, articles, tags newsreader like way. The UI is utilizing Twitter's Bootstrap and jQuery's template engine.

Planets

Local database management through the WebSQL provider, in a fancy representative HTML5 app. It uses new features of CSS3 like keyframe-animations, transitions and transformations.

Advanced RSS using custom provider

Implementing a custom provider to process RSS feeds and extending the context to include custom RSS fields. XML to typed JSON conversion handled by the $data.ModelBinder class.

Form Validation

Our form validator extensions will allow you to apply the constraints defined on the model to validate forms through jQuery validators.

TodoList

Use the same code for multiple providers! See this simple todo-list example running with WebSQL or IndexedDB.
